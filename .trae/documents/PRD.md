# 随机选曲工具 - 产品需求文档

## 1. 产品概述

一款基于React的随机选曲工具，用于帮助用户从曲库中随机选择歌曲。适用于音乐游戏、 playlist 生成、DJ选曲等场景。

主要特点：横排模块化列表展示，包含歌曲封面、名称、难度、作者等完整信息，支持多难度等级筛选。

## 2. 核心功能

### 2.1 功能模块

1. **随机选曲模块**
   - 点击按钮随机从曲库选择歌曲
   - 可选择难度等级筛选（EXPERT、MASTER、Re:MASTER）
   - 展示选中歌曲的完整信息

2. **歌曲列表模块**
   - 横排模块化卡片布局
   - 每张卡片包含：封面、歌曲名、难度等级、作者、难度作者
   - 难度背景素材占位符展示

3. **难度筛选模块**
   - EXPERT难度筛选
   - MASTER难度筛选
   - Re:MASTER难度筛选
   - 支持多选

## 3. 歌曲卡片信息结构

| 字段 | 描述 |
|------|------|
| 歌曲名称 | 歌曲的标题 |
| 难度等级 | EXPERT / MASTER / Re:MASTER |
| 封面 | 歌曲封面图片 |
| 作者 | 曲目原作者 |
| 难度作者 | 谱面难度设计者 |
| 难度背景 | 难度专属背景素材（占位符） |

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：暗色系音乐游戏风格，霓虹灯点缀
- **配色**：
  - 主色：深紫色 `#1a1a2e`
  - 强调色：霓虹粉 `#ff2e63`
  - 次强调色：电光蓝 `#08d9d6`
  - 背景：`#0f0f1a`
- **字体**：
  - 标题：Orbitron（科技感）
  - 正文：Rajdhani
- **布局**：横向滚动卡片列表，卡片宽度固定，高度自适应

### 4.2 卡片设计

- 卡片尺寸：280px × 380px
- 封面区域：顶部 60%
- 信息区域：底部 40%
- 难度标识：右上角浮动标签
- 难度背景：EXPERT（红）、MASTER（蓝）、Re:MASTER（紫）渐变

### 4.3 响应式设计

- 桌面端优先
- 移动端横向滚动

## 5. 难度配色

| 难度 | 背景色 |
|------|--------|
| EXPERT | 红色渐变 `#ff4444 → #ff6b6b` |
| MASTER | 紫色渐变 `#9b59b6 → #e056fd` |
| Re:MASTER | 浅紫色渐变 `#a855f7 → #c084fc` |

## 5. 技术方案

- **框架**：React 18 + Vite
- **样式**：Tailwind CSS
- **状态管理**：React useState/useReducer
- **动画**：CSS 过渡动画
